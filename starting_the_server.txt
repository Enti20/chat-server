1.1 Запуск сервера
1. Перейдите в папку с проектом и выполните:
go run main.go
2. Сервер запустится на порту 8096:
Сервер запущен на :8096
1.2 Запуск HTML-файла
1. Установите локальный HTTP-сервер (если у вас Node.js):
npm install -g http-server
2. Запустите HTTP-сервер в папке с проектом:
http-server
3. Откройте браузер и перейдите по адресу:
                http://localhost:8080
