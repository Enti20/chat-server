Запуск HTML-файла
1. Установите локальный HTTP-сервер (если у вас есть Node.js, если он отсутствует то необходимо установить):
npm install -g http-server

2. Запустите HTTP-сервер в папке с проектом:
http-server
/*данные команды необходимо выполнять в папке проекта. можно использовать терминал вашего IDE
http-server можно невыполнять если проверить что проект работает командой go run main.go
*/

3. Откройте браузер и перейдите по адресу:
                http://localhost:9000
Запуск сервера
1. Перейдите в папку с проектом и выполните:
go run main.go

2. Сервер запустится на порту 9000:
Сервер запущен на :9000


По факту он запустился по ссылке http://localhost:63342/web-server/index.html?_ijt=q1en4vfcu07uvhtvdrktbih0ib&_ij_reload=RELOAD_ON_SAVE
В терминале где запускается через go run main.go работу сервера завершается через Ctrl+C
